## linux命令篇
-------------------------------------------------
### 运行jar包

运行jar包，指定端口为80
java -Dserver.port=80 -jar  *****.jar

后台运行jar包
nohup java -jar *****.jar > /root/logs/a.log &

实时查看日志
tail -f a.log
